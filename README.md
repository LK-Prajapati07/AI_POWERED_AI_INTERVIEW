⚙️ Installation
1️⃣ Clone Repository
git clone https://github.com/LalitP2007/Frontend.git
cd AI-Interview
2️⃣ Install Dependencies
Backend
cd backend
npm install
Frontend
cd frontend
npm install
3️⃣ Environment Variables

Create .env in backend:

PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
CLIENT_URL=http://localhost:5173
OPENAI_API_KEY=your_api_key
4️⃣ Run Project
Backend
npm run dev
Frontend
npm run dev
