# ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/LalitP2007/Frontend.git
```

```bash
cd AI-Interview
```

---

### 2️⃣ Install Dependencies

Backend

```bash
cd backend
npm install
```

Frontend

```bash
cd frontend
npm install
```

---

### 3️⃣ Environment Variables

Create `.env` file inside backend.

```env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
CLIENT_URL=http://localhost:5173
OPENAI_API_KEY=your_api_key
```

---

### 4️⃣ Run Application

Backend

```bash
npm run dev
```

Frontend

```bash
npm run dev
```
